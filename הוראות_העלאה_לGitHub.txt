🚀 הוראות העלאה ל-GitHub - פשוט ומהיר!

📋 מה לעשות:

1️⃣ פתח GitHub Desktop
   - לחץ על כפתור Start
   - חפש "GitHub Desktop"
   - לחץ עליו

2️⃣ צור Repository חדש
   - לחץ על "Create a new repository"
   - שם: business-cards-api
   - בחר תיקיה חדשה (לא זו הנוכחית!)
   - לחץ "Create repository"

3️⃣ העתק קבצים
   העתק מהתיקיה הנוכחית לתיקיה החדשה:
   
   📁 תיקיות:
   - controllers/
   - middleware/
   - models/
   - routes/
   - utils/
   
   📄 קבצים:
   - app.js
   - package.json
   - package-lock.json
   - README.md
   - postman-examples.md
   - .gitignore

4️⃣ Commit ו-Push
   - ב-GitHub Desktop תראה את הקבצים
   - כתוב: "Initial commit: Business Cards API"
   - לחץ "Commit to main"
   - לחץ "Publish repository"
   - בחר Public
   - לחץ "Publish repository"

5️⃣ קבל קישור
   - GitHub Desktop יפתח את הדף ב-GitHub
   - העתק את הקישור מהדפדפן
   - הקישור יהיה: https://github.com/שם-המשתמש/business-cards-api
